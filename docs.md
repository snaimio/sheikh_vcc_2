This project is for my Versio Control and Collaboration assignment.

## Getting Started
To view this project locally, simply clone the repository and open the `index.html` file in any modern web browser.

## Features
* **Interactive Header**: Click the main "Hello World!" heading to trigger a JavaScript alert.
* **Responsive Design**: The CSS ensures the header background image scales correctly using `object-fit: cover`.
* **Organized Structure**: Assets, styles, and scripts are separated into dedicated folders for better maintainability.

## Development Workflow
This project follows a strict feature-branch workflow:
1. Create an issue on GitHub.
2. Checkout a new local branch: `git checkout -b dev-feature-name`.
3. Perform changes and push: `git push origin dev-feature-name`.
4. Open a Pull Request and perform a code review before merging to `main`.
